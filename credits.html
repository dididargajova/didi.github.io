Bootstrap Components & Features Used

This portfolio website was built using Bootstrap 5 and custom CSS, without any templates. Below is a list of the main Bootstrap components and features utilized, along with references:

Navigation & Scrollspy

The site uses a responsive fixed-top navbar with a collapsible menu for smaller screens:contentReference[oaicite:0]{index=0}. Scrollspy is enabled on the body element to update the active navigation link as the user scrolls:contentReference[oaicite:1]{index=1}. We used data-bs-spy="scroll" and data-bs-target="#navbar-main" attributes per Bootstrap's documentation:contentReference[oaicite:2]{index=2}, and initialized it via JavaScript.

Cards

Bootstrap Cards were used to display content in sections like Education, Projects, and Hobbies. Cards provide a flexible content container with padding and contextual backgrounds:contentReference[oaicite:3]{index=3}. We utilized the .card and .card-body classes for structuring card content:contentReference[oaicite:4]{index=4}, and used dark backgrounds to fit the site's theme.

Carousel

A Bootstrap Carousel showcases multiple projects in a slideshow format. The carousel cycles through content (project highlights) with controls and indicators:contentReference[oaicite:5]{index=5}. One slide is marked with .active to initialize the carousel display:contentReference[oaicite:6]{index=6}, and unique IDs and matching data-bs-target attributes link the controls/indicators to the carousel:contentReference[oaicite:7]{index=7}.

Modal & Forms

The Contact section’s "Contact Me" button triggers a Bootstrap Modal containing contact information and a contact form. The modal component provides a popup dialog over the page:contentReference[oaicite:8]{index=8}. Inside, we used a simple form (with <form>, inputs, labels, and a submit button) styled with Bootstrap form classes for consistency. We also included a btn-close with the btn-close-white class for a white close icon on a dark modal.

Tabs

Within the contact modal, we implemented Bootstrap Tabs to toggle between the contact info panel and the message form. Using a <ul class="nav nav-tabs"> element, we created two tabs controlled by Bootstrap’s tab plugin:contentReference[oaicite:9]{index=9}. Each tab is associated with a .tab-pane containing the respective content (info or form), allowing a clean separation of content within the modal.

Bootstrap Grid & Responsiveness

The layout is made responsive using Bootstrap’s grid system. We employed .row and .col-* classes (e.g., .col-md-6) to organize content into columns that stack on smaller devices and sit side-by-side on larger screens:contentReference[oaicite:10]{index=10}. We also used Bootstrap’s utility classes (such as margin/padding utilities and flexbox helpers) to ensure proper spacing and alignment according to modern UX best practices.

Custom CSS & Animations

Some custom CSS was added for a personalized dark theme and interactive feel. For example, we enabled smooth scrolling for in-page links via the scroll-behavior: smooth property. Navbar links change color on hover for better interactivity, and cards lift slightly on hover using a CSS transform transition. These subtle enhancements provide feedback to users and make the page more engaging, aligning with contemporary UX design principles.
